### Installation

1. Clone the repo to your local machince.
   ```sh
   git clone https://github.com/topcowmoo/verschaffeltia-splendida
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the server from the terminal
   ```sh
   npm run start
   ```
